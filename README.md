# SkyCast Weather App

SkyCast is a responsive weather application that provides real-time weather information through a clean and user-friendly interface.

The application allows users to search for cities and instantly receive weather details such as temperature, humidity, wind speed, and current weather conditions.

---

## Overview

SkyCast was developed as a frontend project to demonstrate API integration, responsive design principles, and dynamic user interfaces.

The project combines modern web technologies with real-time weather data to create an interactive and practical weather dashboard.

## Application Workflow

1. User opens the application.
2. User enters a city name.
3. The application sends a request to the weather API.
4. Weather data is retrieved in real time.
5. Results are displayed dynamically.
6. Weather icons change automatically based on current conditions.

## Weather Information Displayed

### Temperature
